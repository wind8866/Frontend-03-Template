## 字符串分析算法
- 字典树
    - 大量高重复字符串的存储与分析
    - 精确匹配，是否相等
    - 用在搜索关键词
- KMP
    - 在长字符串里找短字符串
- Wildcard
    - 带通配符的字符串模式
- 正则
    - 字符串通用模式匹配
- 状态机
    - 通用的字符串分析
- LL、LR
    - 字符串多层级解构分析


## 字典树
字典树时哈希树的一种特例，哈希树在字符串领域里面最直接的应用体现就是字典树。


## KMP字符串模式匹配算法
```
abcdabce
   j
abcdabcdabcex
       i
```
1. 找出字符串的自重复行为：每次的开头都去掉一位，检查剩下的开头有没有重复的
2. 如何用数据解构描述上述状态，只看字符串的开头是否匹配

```
abcdabce
00000123
数字2表示前面已经有两个（ab）是重复的（所有的匹配都是从头开始），如果当前位置不匹配，则跳回位置2
```
